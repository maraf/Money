New features

  • Yearly income overview page.
  • Income search page.
  • Calendar picker replaces the date box when selecting an expense date.
  • Submitting a zero amount expense skips a scheduled expense for the period.
  • Context menu to change an expense on the Monthly checklist.
  • Pass the 'Expected when' value when creating an expense from the checklist.
  • Use keyboard arrows to navigate in the category picker.
  • Templates search filter is preserved in browser history.
  • Reduced placeholders on Overview for periods in the future.
  • Form is disabled while submitting to prevent double submission.

Fixed bugs

  • Multiple menu items highlighted for settings and password.
  • Overview applied an expense that didn't fit the current category.
  • Currency was reset to default when changing the amount.
  • Couldn't enter an expense with decimals when display was set to hide decimals.
  • Couldn't reset the amount on an existing expense template.
